HTML

  • The head element contains information about the webpage.
  • The meta element contains information not visible to the user that is used by the web browser
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changed files in our working directory to the staging area of GIT ready to be commited
  • git commit -m "": commits our added codes to our git repo
  • git pull origin main: to sync our local repo with our github repo. NOTE: its important to pull main into feature before we push
  • git push origin feature/"name": pushes all changes to our github repo

JavaScript

  • js scripts are linked just before the end of the body tag using the script tags
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
